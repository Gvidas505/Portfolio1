Projects

Integrated Study and Research Repository with Flashcard-Based Learning

The Integrated Study and Research Repository with Flashcard-Based Learning is a database-driven application designed to help students organise academic materials and improve knowledge retention. The system allows users to store and manage research sources in a structured repository while linking key concepts to interactive flashcards for active recall practice. By combining information storage with spaced-repetition learning techniques, the project aims to improve both organisation of study materials and long-term understanding. The application demonstrates skills in database design, SQL, and system development, focusing on creating an efficient and user-friendly learning tool for students.
